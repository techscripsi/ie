Запуск из командной строки

Вы можете запустить Internet Explorer из командной строки.

Чтобы запустить Internet Explorer из командной строки, выполните следующие действия.
  1. В левом нижнем углу экрана в строке поиска рядом со значком (Пуск) введите cmd (см. изображение ниже).
  2. Наклавиатуре нажмите на клавишу ВВОД.
    Отобразится окно командной строки (см. изображение ниже).
  3. Введите команду start iexplore.exe и нажмите клавишу ВВОД (см. изображение ниже).
    Окно браузера отобразится на рабочем столе.
