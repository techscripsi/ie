Запуск с помощью строки поиска

Вы можете запустить Internet Explorer с помощью строки поиска.

Чтобы запустить Internet Explorer с помощью строки поиска, выполните следующие действия.

  1. В левом нижнем углу экрана в строке поиска рядом со значком (Пуск) и начните вводить Internet Explorer.
    В меню Пуск отобразится соответствующий пункт (см. изображение ниже).
  2. Щёлкните на пункте Internet Explorer.
    Окно браузера отобразится на рабочем столе.
