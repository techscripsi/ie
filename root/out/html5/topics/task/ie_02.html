Запуск из меню Пуск

Вы можете запустить Internet Explorer, найдя браузер в меню Пуск.

Чтобы запустить Internet Explorer из меню Пуск, выполните следующие действия.
  1. Щёлкните на значке (Пуск) в левом нижнем углу экрана.
    Отобразится содержимое меню Пуск (см. изображение ниже).
  2. В списке меню слева выберите Стандартные - Windows > Internet Explorer.
    Окно браузера отобразится на рабочем столе.
