Настройка быстрого запуска

Помимо способов запуска Internet Explorer, которые доступны в операционной системе Windows 10 по умолчанию, существует возможсность настроить быстрые способы запуска Internet Explorer:
  • с начально экрана Windows (или из области быстрого запуска меню Пуск);
  • из панели задач Windows.
  • Закрепление значка Internet Explorer на начальном экране
    Вы можете закрепить значок запуска Internet Explorer на начальном экране Windows (или в области быстрого запуска меню Пуск).
  • Закрепление значка Internet Exporer на панели задач
    Вы можете закрепить значок Internet Explorer на панели задач.
