Варианты запуска

Запустить Internet Explorer можно несколькими способами. В настоящем разделе представлены процедуры, следуя которым вы сможете запустить Internet Explorer на компьютере под управлениям Windows 10.

  • Запуск с помощью строки поиска
    Вы можете запустить Internet Explorer с помощью строки поиска.
  • Запуск из меню Пуск
    Вы можете запустить Internet Explorer, найдя браузер в меню Пуск.
  • Запуск из командной строки
    Вы можете запустить Internet Explorer из командной строки.
